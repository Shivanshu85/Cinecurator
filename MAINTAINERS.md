## Becoming a Maintainer

We welcome long-term contributors to join the maintainer team! To become a maintainer:

1. Demonstrate consistent, high-quality contributions across frontend, ML backend, or documentation.
2. Provide thoughtful code reviews on community Pull Requests.
3. Help triage issues and assist community members in GitHub Discussions.
4. Adhere strictly to project quality standards and the [Code of Conduct](CODE_OF_CONDUCT.md).
