## When to Open a GitHub Issue

Please open a GitHub issue if:
1. You have encountered a bug or unexpected behavior that can be reproduced using current `main`.
2. You have verified that the issue is not caused by missing environment variables (`.env.local`).
3. You have searched existing [GitHub Issues](https://github.com/Shivanshu85/Cinecurator/issues) and confirmed it is not a duplicate.
