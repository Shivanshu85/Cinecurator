# Acknowledgements & Credits

CineCurator is built on top of incredible open-source technology, public datasets, and developer tools. We express our gratitude to the creators and maintainers of the projects listed below.

---

## 🎬 Datasets & Media APIs

- **[The Movie Database (TMDB):](https://www.themoviedb.org/)** Provides rich movie metadata, poster backdrops, cast information, and trailer video key indexing.
- **[OMDb API:](http://www.omdbapi.com/)** Provides IMDb rating enrichment and plot summaries.
- **[YouTube Data API:](https://developers.google.com/youtube/v3)** Provides fallback video trailer lookup services.

---

## 🛠️ Frameworks & Open Source Libraries

### Frontend Ecosystem
- **[Next.js (Vercel):](https://nextjs.org/)** React framework powering server-side rendering, API routes, and App Router architecture.
- **[React 18:](https://react.dev/)** UI component library.
- **[Tailwind CSS:](https://tailwindcss.com/)** Utility-first CSS framework.
- **[TanStack React Query:](https://tanstack.com/query/latest)** Asynchronous data fetching, caching, and state synchronization.
- **[Zustand:](https://github.com/pmndrs/zustand)** Lightweight client state management.
- **[Supabase JS Client:](https://supabase.com/docs/reference/javascript/introduction)** PostgreSQL client and authentication provider.

### Machine Learning & Backend Ecosystem
- **[FastAPI:](https://fastapi.tiangolo.com/)** Modern, fast web framework for building Python APIs.
- **[Scikit-Learn:](https://scikit-learn.org/)** Machine learning library powering TF-IDF vectorization and cosine similarity calculations.
- **[NumPy & Pandas:](https://numpy.org/)** High-performance mathematical computing and data manipulation libraries.
- **[Uvicorn:](https://www.uvicorn.org/)** Lightning-fast ASGI server implementation.
